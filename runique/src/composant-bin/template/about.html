🦀 Rusti Framework

A modern, Django-inspired web framework built with Rust's power and safety.

✨ Key Features

  • Strongly typed forms and validation
  • Django-like project structure
  • Built on Axum, Tera, and SeaORM
  • First-class async support with Tokio

Why Rusti?

Rusti brings the elegant architecture patterns of Django to the Rust ecosystem. If you love Django's “batteries included” philosophy but want Rust’s performance and type safety, Rusti is for you.
