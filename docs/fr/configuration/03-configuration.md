# Configuration

Toute la configuration est facilitée via `.env` et est chargée dans une struct `RuniqueConfig`.

| [Builder](https://github.com/seb-alliot/runique/blob/main/docs/fr/configuration/builder/builder.md) | Builder classique, Builder Intelligent, méthodes, valeurs par défaut |
