Configuration complète d'une application Runique via le builder — tout est déclaratif, typé, et validé au démarrage.

main.rs — configuration minimale
#[tokio::main]
async fn main() -> Result<(), AppError> {
    init_logging();

    let db_config = DatabaseConfig::from_env()?.min_connections(2).build();
    let db:DatabaseConnection = db_config.connect().await?;


    RuniqueApp::builder()
        .with_database(db)
        .routes(url::.routes()())
        .statics()
        .build()?
        .run().await
}
Sécurité — middlewares via builder
RuniqueApp::builder()
    // Rate limiter : 100 req/60s par IP
    .with_rate_limiter(
        RateLimiter::new()
            .max_requests(100)
            .window_secs(60)
    )
    // Hôtes autorisés
    .with_allowed_hosts(vec![
        "monsite.fr",
        "www.monsite.fr",
    ])
    // CSP — profil strict
    .with_csp(Csp::strict())
    // Redirection HTTPS + HSTS
    .with_https_redirect(true)
