Prérequis, configuration et commandes CLI pour démarrer un projet Runique.

Prérequis
# Rust 1.85+
$ rustup update stable
$ rustc --version

# SeaORM CLI (migrations)
$ cargo install sea-orm-cli

# CLI Runique
$ cargo install runique-cli
.env minimal
# Serveur
IP_SERVER=127.0.0.1
PORT=3000
DEBUG=true

# Sécurité
SECRET_KEY=changez-moi-en-production

# Base de données (SQLite dev)
DB_ENGINE=sqlite
DB_NAME=local_base.sqlite

    
Lancer la démo

    
$ git clone https://github.com/seb-alliot/runique.git
$ cd runique
$ cargo run -p demo-app
Commandes CLI
# Créer un nouveau projet
$ runique new mon-projet

# Générer le panneau admin depuis src/admin.rs
$ runique start

# Créer un superutilisateur admin
$ runique create-superuser

# Générer les migrations
$ runique makemigrations \
    --entities src/entities \
    --migrations migration/src

# Appliquer / rollback / statut
$ runique migration up
$ runique migration down
$ runique migration status
.env production (PostgreSQL)
DEBUG=false
SECRET_KEY=clé-aléatoire-longue

DB_ENGINE=postgres
DB_USER=monuser
DB_PASSWORD=secret
DB_HOST=localhost
DB_PORT=5432
DB_NAME=mabase
