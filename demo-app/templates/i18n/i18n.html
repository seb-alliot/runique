{{ title }}

Internationalisation des templates internes de Runique.
N'affecte que les données renvoyées par le framework lui-même (pages d'erreur, admin, messages de validation).

Déclaration dans main.rs
use runique::prelude::*;

// À appeler une seule fois au démarrage, avant builder::new().
set_lang(Lang::Fr);
Langues disponibles
Lang::Fr  // Français
Lang::En  // English  (défaut si non appelé)
Lang::It  // Italiano
Lang::Es  // Español
Lang::De  // Deutsch
Lang::Pt  // Português
Lang::Ja  // 日本語
Lang::Zh  // 中文
Lang::Ru  // Русский
Ce que ça traduit
// Pages d'erreur Runique
404 Not Found  →  "Page introuvable"
429 Too Many    →  "Trop de requêtes"
500 Server Err  →  "Erreur interne"

// Messages de validation des formulaires
required  →  "Ce champ est obligatoire"
max_len   →  "Trop long (max : 255)"

// Interface d'administration
"Enregistrer" / "Supprimer" / "Rechercher" …
Ce que ça ne traduit PAS
// Vos propres templates Tera → votre responsabilité.
// Vos propres messages d'erreur métier → votre responsabilité.
// La langue ne change pas dynamiquement par requête.
// C'est un réglage global au démarrage du serveur.
