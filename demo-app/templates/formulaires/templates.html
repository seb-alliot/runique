Deux façons de rendre un formulaire dans un template Tera — rendu automatique ou champ par champ.

Rendu automatique — tous les champs d'un coup
{# views.rs — injecter le formulaire dans le contexte #}
context_update!(request => {
    "mon_form" => &form,
});

{# template.html #}
<form method="post" action="{% raw %}{%{% endraw %} link 'ma_route' {% raw %}%}{% endraw %}">
    {% raw %}{%{% endraw %} form.mon_form {% raw %}%}{% endraw %}
    <!-- CSRF injecté automatiquement avant le 1er champ -->
    <!-- Tous les champs rendus dans l'ordre de déclaration -->
    <button type="submit">Envoyer</button>
</form>
Comportement automatique
{# {% form.mon_form %} fait tout ça d'un coup : #}

{# 1. Injecte le token CSRF avant le 1er champ #}
{# 2. Rend chaque champ dans l'ordre de déclaration #}
{# 3. Injecte les scripts JS après le dernier champ #}

Rendu champ par champ
<form method="post">
    <div class="col-left">
        {% raw %}{%{% endraw %} form.mon_form.username {% raw %}%}{% endraw %}
        {% raw %}{%{% endraw %} form.mon_form.email {% raw %}%}{% endraw %}
    </div>

    <div class="col-right">
        {% raw %}{%{% endraw %} form.mon_form.password {% raw %}%}{% endraw %}
        {% raw %}{%{% endraw %} form.mon_form.bio {% raw %}%}{% endraw %}
    </div>

    <!-- CSRF injecté automatiquement sur le 1er champ -->
    <button type="submit">Envoyer</button>
</form>
