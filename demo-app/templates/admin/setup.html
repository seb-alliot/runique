Installer et démarrer le panneau admin via la CLI Runique.

1. Déclarer dans src/admin.rs
admin! {
    users: users::Model => RegisterForm {
        title: "Utilisateurs",
        permissions: ["admin"]
    }
}
2. Générer le panneau
# Génère src/admins/admin_panel.rs
$ runique start
3. Brancher dans main.rs
builder::new(config)
    .with_admin(|a| {
        a.site_title("Administration")
         .auth(RuniqueAdminAuth::new())
         .routes(admins::routes("/admin"))
         .with_state(admins::admin_state())
    })
Ce que runique start génère
