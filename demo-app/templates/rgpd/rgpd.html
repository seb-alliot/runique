Demo site

This site is a technical demonstration of the Runique framework. It has no commercial purpose. The data entered is used solely to illustrate how the framework works.

Data collected

When creating an account, the following information is recorded:

Username, email address, password (hashed, never stored in plain text).

Since this is a demonstration, no real email is required — a fake address (e.g.: test@example.com) is perfectly acceptable.

This data is stored in a local database for testing purposes only.

Data usage

Your data is never:

— shared with third parties

— used for commercial purposes

— used for profiling or marketing

Data retention

Data can be deleted at any time upon request. The demo database is reset periodically.

Contact

For any deletion request or question, open an issue on the GitHub repository.
