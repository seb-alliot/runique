## [0.1.0] - 2024-XX-XX

### Added
- Initial release of Rusti framework
- Django-inspired application structure
- Axum-based routing and middleware
- Tera templating engine integration
- Optional SeaORM database support
- Configuration management via Settings struct
- Built-in error handling and debug pages
- Static and media file serving
- Session management
- Comprehensive examples and documentation

### Features
- `RustiApp` builder pattern for easy setup
- Environment-based configuration
- Automatic template reloading in debug mode
- Graceful shutdown handling
- Type-safe response helpers
- Middleware system for cross-cutting concerns
